event.result

event.result Returns: Object

Description: This attribute contains the last value returned by an event handler that was triggered by this event, unless the value was undefined.

  • version added: 1.3event.result

Example:

Alert previous handler's return value

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-latest.js"></script>
</head>
<body>
	
<script>$("p").click(function(event) {
  return "hey"
});
$("p").click(function(event) {
  alert( event.result );
});  </script>
</body>
</html>

Demo:

Result:

"hey"
