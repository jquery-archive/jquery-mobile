:animated Selector

animated selector

version added: 1.2jQuery(':animated')

Description: Select all elements that are in the progress of an animation at the time the selector is run.

Example:

Change the color of any div that is animated.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background:yellow; border:1px solid #AAA; width:80px; height:80px; margin:5px; float:left; }
  div.colored { background:green; }
  </style>
  <script src="http://code.jquery.com/jquery-latest.js"></script>
</head>
<body>
	<button id="run">Run</button>

  <div></div>
  <div id="mover"></div>
  <div></div>
<script>

    $("#run").click(function(){
      $("div:animated").toggleClass("colored");
    });
    function animateIt() {
      $("#mover").slideToggle("slow", animateIt);
    }
    animateIt();
</script>
</body>
</html>
