To add a color picker widget to your page, add a div with the attribute data-role="hsvpicker" set:


<div data-role="hsvpicker"></div>

This gives you an HSV color picker:

You can also enhance your color inputs with this widget. To do this, modify the initSelector option during the mobileinit event:

$( document ).bind( "mobileinit", function(){
   $.mobile.hsvpicker.prototype.options.initSelector += ", input[type='color']";
});

Calling the hsvpicker plugin

This plugin will auto initialize on any page that contains a div with the data-role="hsvpicker" attribute. However, if needed you can directly call the hsvpicker plugin on any selector, just like any jQuery plugin:


$('input').hsvpicker();

Here's how you can add an hsvpicker-enhanced color input to your form:


<script>
$(document).bind("pagecreate", function() {$("#neededColor").hsvpicker();});
</script>
<form action="#" method="get">
	<div data-role="fieldcontain">
		<label for="#neededColor">Choose a color:</label>
		<input id="neededColor" type="color" value="#112233"></input>
	</div>
</form>
