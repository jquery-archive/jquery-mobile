To add a color picker to your page, use a standard input with the type="color" attribute. The input's value is used to configure the initial color. The framework will parse these attributes to configure the color picker widget.

When you press close on the color picker popup window, the framework will update the native input's value (and vice-versa) so they are always in sync; this ensures that the value is submitted with the form.

Set the for attribute of the label to match the ID of the input so they are semantically associated and wrap them in a div with the data-role="fieldcontain" attribute to group them.


<div data-role="fieldcontain">
   <label for="colorpickerbutton">Color picker button:</label>
   <input type="color" name="colorpickerbutton" id="colorpickerbutton" value="#422442" />
</div>

Calling the colorpickerbutton plugin

This plugin will auto initialize on any page that contains a text input with the type="color" attribute. However, if needed you can directly call the colorpickerbutton plugin on any selector, just like any jQuery plugin:


$('input').colorpickerbutton();
