Overlay panels

Taking customization further, here is an example of a popup that has been customized to look like a vertical panel with three mini buttons:

Open panel

Here is the HTML markup for the button and panel:


<a href="#popupPanel" data-rel="popup" data-transition="slide" data-position-to="window" data-role="button">Panel right</a>
			
<div data-role="popup" id="popupPanel" data-corners="false" data-theme="none" data-shadow="false" data-tolerance="0,0">

    <div class="inner">
	 
        <button data-theme="b" data-icon="back" data-mini="true">Back</button>
        <button data-theme="b" data-icon="grid" data-mini="true">Menu</button>
        <button data-theme="b" data-icon="search" data-mini="true">Search</button>
		  
    </div>
	 
</div>

To style the panel, and attach it to the right edge, the following CSS is used:


#popupPanel-popup {
    width: 90%;
    max-width: 200px !important;
    right: 0 !important;
    left: auto !important;
}
#popupPanel {
    border-left: 1px solid #444;
    margin-left: -1px;
    background: rgba(0,0,0,.6);
    position: relative;
    width: 100%;
    height: 100%;
}
#popupPanel .inner {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    padding: 45px 15px 15px;
}
#popupPanel .inner .ui-btn {
    margin: 0 0 1em 0;
}
